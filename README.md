Construction Company Website

In this project I am building a Construction Company Website! This is a full-stack web application designed for a construction company to showcase their services, projects, and contact information.

React-Vite-Tailwind Project with Express Backend
This is a full-stack web application template built with React, Vite,Typescript, Tailwind CSS for the frontend, and Express.js for the backend.

Features
Frontend built with React, a fast and efficient JavaScript and Typescript library for building user interfaces.
Utilizes Vite for lightning-fast development and build times.
Tailwind CSS for rapid UI development with utility-first CSS framework.
Express.js for building robust and scalable backend APIs.
Nodemon for automatically restarting the server when file changes are detected.
Nodemailer for sending emails from the backend.
Prerequisites
Node.js installed on your machine.
npm or yarn package manager.
